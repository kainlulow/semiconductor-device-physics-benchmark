# Code of Conduct

This project follows a simple standard: be respectful, technically constructive, and precise.

Participants should:

- focus criticism on ideas, data, and implementation details;
- assume good faith while asking for evidence;
- avoid harassment, personal attacks, and exclusionary language;
- help keep discussions useful for researchers, engineers, students, and maintainers.

Maintainers may moderate issues, pull requests, or comments that do not meet this standard.
